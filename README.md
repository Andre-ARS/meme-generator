# Projeto Meme Generator

# Contexto
Este projeto trata-se de um gerador de memes utilizando javaScript com manipulação do DOM. Nele o usuário é capaz de escolher uma imagem já salva no site ou uma de seu próprio dispositivo, escrever uma frase e escolher uma borda diferente.

## Tecnologias usadas

> Desenvolvido usando: CSS3, HTML5, JavaScriptES6

## Testando Localmente

> Clone o Repositório
```bash
git clone git@github.com:Andre-ARS/meme-generator.git
``` 
> Dentro do diretório do projeto, instale as dependencias
```bash
npm install
``` 
Link do site com Deploy utilizando Vercel: [Meme Generator](https://meme-generator-ars.vercel.app/)
